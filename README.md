# Form Project

Простая анкета на PHP с сохранением данных в MySQL.

1. Установите [XAMPP](https://www.apachefriends.org/) или другой локальный сервер с поддержкой PHP и MySQL.
2. Склонируйте репозиторий в папку `htdocs` (для XAMPP)
3. Импортируйте базу данных:
   Создайте новую базу, например formtest
4. Настройте подключение к базе в db.php
5. Перейдите в браузере по адресу http://localhost/form_project/index.php
